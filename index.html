Speed up your GoPro Videos.

Use this BASH script to process your GoPro videos on the command line. This script uses ffmpeg, mplayer, mjpegtools(optional), youtube-upload(optional):

Usage: timex -n [name] -r [rate] -m -s -y -h
    -n  New name for file, default is the date + rate
    -r  Rate to speed up video, default is 24
    -m  Use mjpeg tools to compress video
    -s  silent - do not include audio
    -y  Upload to Youtube
        Program will ask for name, description, user and password
    -h  This help message

    Tools needed to run this script:    
            ffmpeg  (Video I/O)
            mjpegtools  (Video Compression)
            mplayer  (Audio Compression)
            youtube-upload  (Command Line upload to Youtube)

Required -vs- Optional

ffmpeg does the heavy lifting in this script. mplayer is used to compress the audio. The resulting file's quality improves and render time decreases if you add mjpegtools. youtube-upload is only needed if you want the script to automatically upload the final file up to youtube.

Moving Forward

I'm still working on ways to improve the quality of video compression using just ffmpeg. I also plan to add jpg to video functionality to the script in the near future.
